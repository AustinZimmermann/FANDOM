# ババ

<img src="https://static.wikia.nocookie.net/fzero/images/e/e2/Baba.jpg" height="350">

**ババ**（Baba）は、任天堂のレースゲーム『F-ZERO』シリーズに登場するキャラクター（F-ZEROパイロット）である。

## 概要

初出はNINTENDO64ソフト『F-ZERO X』。かつて「神の声」を聞いたとされる神秘的な出自を持つパイロットであり、野生的な直感と洗練されたダンススキルをレースに活かしている。

かつては大自然の中で生活していたが、スカウトされてF-ZEROパイロットへと転身した。彼のダンスのようなドライビングテクニックは、観客を魅了するだけでなく、他のパイロットを翻弄する一因にもなっている。

## プロフィール

  * **性別**: 男
  * **年齢**: 19歳（X時点）/ 20歳（GX時点）
  * **職業**: F-ZEROパイロット / ダンサー
  * **ナンバー**: 04
  * **搭乗機**: **アイアンタイガー** (IRON TIGER)

## 搭乗マシン：アイアンタイガー

ババが操るナンバー04のマシン。

| 性能項目 | 評価 |
| :--- | :--- |
| **ボディ** | B |
| **ブースト** | C |
| **グリップ** | A |

### 特徴

グリップ性能が非常に高く、急なカーブでも安定した走行が可能である。旋回性能に優れるため初心者にも扱いやすい部類に入るが、最高速度やブースト性能は平均的であり、直線主体のコースでは苦戦を強いられる傾向にある。

## 劇中での活躍

  * **F-ZERO X**: 初登場。若き天才ダンサーとしての側面が強調された。
  * **F-ZERO GX**: 優勝賞金でダンススクールを建設しようという野望を持って参戦。ムービーシーンでは、優勝後に自慢のダンスを披露する姿が描かれている。
  * **F-ZERO ファルコン伝説**: アニメ版およびそのゲーム化作品にも登場。ダークリオン（ブラック シャドー）の配下として活動する場面も見られるが、基本的には独自の美学を持つキャラクターとして描かれた。

## 余談

  * 彼の名前「ババ」は、開発スタッフによれば、当時流行していたダンスや音楽シーンのニュアンスが含まれているとされる。
  * ナンバー「04」は一般的に不吉な数字とされるが、彼の場合はそれを逆手に取ったカリスマ性を演出している。
